REST APIs for V1 pools on SaucerSwap

Access the SaucerSwap API using the following base URLs for testnet and mainnet environments:



| Hedera Network | Base URL                                                                              |
| -------------- | ------------------------------------------------------------------------------------- |
| Mainnet        | [https://api.saucerswap.finance/](https://api.saucerswap.finance/v2/pools/)           |
| Testnet        | [https://test-api.saucerswap.finance/](https://test-api.saucerswap.finance/v2/pools/) |
| Previewnet     | *Not supported*                                                                       |



### Get the latest candlestick data for pool by id

get

/pools/conversionRates/latest/*\{poolId}*

Test it

Path parameters

poolIdintegerrequired

Pool ID

Example: `1`

Query parameters

intervalstring 路 enumrequired

Data interval

Example: `HOUR`

Options: `FIVEMIN`, `HOUR`, `DAY`, `WEEK`

invertedboolean 路 default: false

Responses

200

Successful response

application/json

<Tabs />

```
curl -L \
  --url '/pools/conversionRates/latest/{poolId}?interval=HOUR'
```

### Get historical candlestick data for pool by id

get

/pools/conversionRates/*\{poolId}*

Test it

Path parameters

poolIdintegerrequired

Pool ID

Example: `1`

Query parameters

fromintegerrequired

From Unix seconds

Example: `1697600000`

tointegerrequired

To Unix seconds

Example: `1697610000`

intervalstring 路 enumrequired

Data interval

Example: `HOUR`

Options: `FIVEMIN`, `HOUR`, `DAY`, `WEEK`

invertedboolean 路 default: false

Responses

200

Successful response

application/json

<Tabs />

```
curl -L \
  --url '/pools/conversionRates/{poolId}?from=1697600000&to=1697610000&interval=HOUR'
```
