REST APIs for SaucerSwap V2 pools and positions

Access the SaucerSwap API using the following base URLs for testnet and mainnet environments:

Hedera Network

Base URL

Mainnet

[https://api.saucerswap.finance/](https://api.saucerswap.finance/v2/pools/)

Testnet

[https://test-api.saucerswap.finance/](https://test-api.saucerswap.finance/v2/pools/)

Previewnet

*Not supported*

### Get all liquidity positions by account id

get

/v2/nfts/*\{accountId}*/positions

Test it

Path parameters

accountIdstringrequired

Hedera account id

Example: `0.0.12345`

Responses

200

Successful response

application/json

<Tabs />

```
curl -L \
  --url '/v2/nfts/{accountId}/positions'
```
