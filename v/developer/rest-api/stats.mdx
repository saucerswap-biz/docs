REST APIs for SaucerSwap statistics.

Access the SaucerSwap API using the following base URLs for testnet and mainnet environments:



| Hedera Network | Base URL                                                                              |
| -------------- | ------------------------------------------------------------------------------------- |
| Mainnet        | [https://api.saucerswap.finance/](https://api.saucerswap.finance/v2/pools/)           |
| Testnet        | [https://test-api.saucerswap.finance/](https://test-api.saucerswap.finance/v2/pools/) |
| Preview        | *Not supported*                                                                       |

### Get historical HBAR price data (minutely)

get

/stats/hbarHistoricalPrices

Test it

Query parameters

fromintegerrequired

From Unix seconds

Example: `1697600000`

tointegerrequired

To Unix seconds

Example: `1697610000`

Responses

200

Successful response

application/json

<Tabs />

```
curl -L \
  --url '/stats/hbarHistoricalPrices?from=1697600000&to=1697610000'
```

<Tabs />

```
[
  {
    "id": 548294,
    "priceUsd": 0.04830379666666667,
    "timestampSeconds": 1697599980
  }
]
```

Successful response

### Get platform historical liquidity or volume data

get

/stats/platformData

Test it

Query parameters

fromintegerrequired

From Unix seconds

Example: `1697600000`

tointegerrequired

To Unix seconds

Example: `1697610000`

intervalstring · enumrequired

Example: `HOUR`

Options: `HOUR`, `DAY`, `WEEK`

fieldstring · enumrequired

Example: `LIQUIDITY`

Options: `LIQUIDITY`, `VOLUME`

Responses

200

Successful response

application/json

<Tabs />

```
curl -L \
  --url '/stats/platformData?from=1697600000&to=1697610000&interval=HOUR&field=LIQUIDITY'
```

<Tabs />

```
[
  {
    "valueHbar": 46701854905030720,
    "timestampSeconds": 1697598000
  }
]
```

Successful response
